Body¶

The agent’s body defines its physical structure. It consists of bones. Each bone has a shape attached.

Adds a new BoneBox to the Body.

This method creates a new BoneBox object with the given id, parentId, and name parameters and adds it to the Body.

See also

BoneBox, Body

Parameters:

  • id – The unique identifier for the new BoneBox.

  • parentId – The unique identifier of the parent BoneBox. Default is 0.

  • name – The name of the new BoneBox. Default is an empty string.

Returns:

A pointer to the newly created BoneBox object.
